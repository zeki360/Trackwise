# **App Name**: TrackWise

## Core Features:

- Issue Reporting: Report maintenance and IT issues with categorization (facility: external, internal; IT: hardware, software, office machines, internet, repair; purchase).
- Role-Based Routing: Automatically route reported issues to the appropriate role: Operations Manager (facility), IT Manager (IT), Purchaser (purchase requests), Admin Director (unresolved issues).
- Status Tracking: Track the status of reported issues (accepted, pending, ongoing, finished).
- Issue Categorization: AI tool to suggest most relevant categories to issues raised to simplify reporting and route the request to most suited team/individual for resolve.
- Homepage Category Listing: List all main categories (Facility, IT, Purchase) on the homepage for easy navigation.

## Style Guidelines:

- Primary color: Blue (#4285F4), conveying trust and efficiency, and suitable for a professional tool.
- Background color: Light gray (#F5F5F5) for a clean, uncluttered interface.
- Accent color: Orange (#FFA726) for highlighting important actions and status indicators. It stands out well against the blue primary.
- Body and headline font: 'PT Sans', a humanist sans-serif for both headings and body text to provide a modern, approachable, and readable experience.
- Use simple, clear icons to represent each category and status. Icons should be easily recognizable and consistent throughout the application.
- Prioritize a clean and intuitive layout. Use clear visual hierarchy to guide users through the application and ensure easy access to key features.
- Incorporate subtle animations, such as progress bar updates during issue resolution or smooth transitions between screens. These add a polished feel.
